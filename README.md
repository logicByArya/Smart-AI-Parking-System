# Smart AI Parking System 🚗

## Overview

Smart AI Parking System is an intelligent parking management solution that combines Artificial Intelligence, Computer Vision, IoT, and Embedded Systems to monitor parking spaces in real time.
The system uses a webcam and YOLO object detection model to identify occupied and vacant parking slots and updates the parking status instantly.
The parking information is displayed through a web application, Firebase database, and an LCD display connected to an ESP-based embedded system, enabling users to easily find available parking spaces.

## Features

* Real-time parking slot monitoring
* YOLO-based vehicle detection
* Automatic occupied/vacant slot identification
* Firebase cloud database integration
* Live parking status updates
* Web-based monitoring dashboard
* LCD display for parking availability
* AI and IoT integration

## System Architecture

1. Webcam captures the parking area.
2. YOLO model detects vehicles in the captured frames.
3. Predefined parking slot coordinates are checked for occupancy.
4. Slot status is updated to Firebase in real time.
5. Web application displays available and occupied slots.
6. LCD display shows parking availability information.
